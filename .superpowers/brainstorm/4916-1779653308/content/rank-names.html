Rank Tier Naming

Eight tiers from beginner to endgame. Which theme feels right for a study app?

A

Classical Scholar

Rooted in academic tradition. Dignified, timeless, not game-y at all.

Wanderer → Seeker → Apprentice → Scholar → Adept → Sage → Master → Luminary
B

Forge & Mastery

Craftsmanship language — knowledge as something built and forged. Feels earnest and grounded.

Raw → Novice → Apprentice → Craftsman → Artisan → Expert → Virtuoso → Grandmaster
C

Light & Ascension

A journey from darkness to light. Poetic, slightly mystical — fits a "becoming" narrative well.

Ember → Spark → Flame → Beacon → Radiant → Brilliant → Ascendant → Eternal
